### 401エラーの原因切り分け（正しく実装できていたことの確認）

投稿・読み取り共に401 Unauthorizedが発生。以下の手順で原因を切り分けた：

1. `diagnoseXKeyFormat()` — 鍵の長さ・空白・ハイフン位置を確認 → **問題なし**（API Key 25文字、API Secret 50文字、Access Token 50文字＋ハイフンあり、Access Token Secret 45文字、いずれも空白なし）
2. `verifyXCredentials()` — 投稿を伴わない読み取り専用API（GET /2/users/me）でテスト → **これも401**（書き込み権限固有の問題ではないと判明）
3. `testOAuthSignatureAlgorithm()` — Twitter公式ドキュメントの既知サンプル値で署名計算を検証 → **完全一致、アルゴリズムは正しい**
4. 鍵は「Consumer Keys」「Access Token and Secret」を同時に再生成して組み合わせのズレも排除済み
