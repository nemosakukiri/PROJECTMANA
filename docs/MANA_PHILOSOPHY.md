# MANA 思想・設計哲学

## PROJECT MANAが信じているもの

私たちは、人には主体性があり、可能性があると信じています。

だから答えを与えるのではなく、観測を残します。

結論を押し付けるのではなく、問いを展示します。

一人ひとりが、自ら考え、自ら気づく力を信じています。

---

## MANAとは何か

MANAはニュースサイトではない。**社会観測ミュージアム**である。

---

## 三層構造

### 第一層：百葉箱

記事・証言・事例を観測する。

気象台が気温を測るように、社会に起きていることをそのまま記録する。
解釈や判断は加えない。まず観測する。

### 第二層：収蔵庫

タグで整理する。

タグは比較・検索・集計のために存在する。
タグは分類の道具であり、意味の道具ではない。

### 第三層：展示室

窓で社会を考える。

**窓はタグの代替ではない。**
窓は「どこから光を当てるか」という考察の入口である。

タグは収蔵庫の整理番号だ。比較・検索・集計のために存在する。
窓は問いだ。来館者が歩きながら考えるための入口として存在する。

「民主主義の窓」とは、公益通報タグの記事一覧でも、情報公開タグの記事一覧でもない。

> 「民主主義とは何か」  
> 「選挙だけで民主主義と言えるのか」  
> 「権力の補正機能はどこにあるのか」

という問いを考えるための展示室だ。

その材料として、兵庫県知事問題・公益通報事案・情報公開請求・住民監査請求・議会や行政の機能不全などが並ぶ。一つの出来事に、どこから光を当てるか。それが窓の役割である。

## MANAの価値観

MANAは結論を押し付けない。

しかし、

**「人は平等で支え合う存在である」**

という価値観を土台として持つ。

原爆資料館が原爆を肯定しないのと同じように——
価値観は持つが、思考は強制しない。

---

## 開発上の含意

この思想は「UIの変更」ではなく**サイト設計の根幹**である。

- 記事収集（第一層）はできるだけ価値中立に広く観測する
- タグ設計（第二層）は比較・集計が可能な粒度で統一する
- 窓（第三層）はタグの組み合わせではなく、問いの設定として設計する

新機能の設計判断に迷ったときは、この三層のどこに位置するかを問うこと。

### 窓を実装するときの問い

- その窓は何を問うているか（問いを先に書く）
- タグ一覧になっていないか（なっていれば窓ではなく収蔵庫の機能）
- 来館者はその窓から何を考えるか

## 展示の思想

### 展示は「答え」ではなく「光の当て方」

同じ事案に、複数の視点から光を当てることができる。  
展示システムはその複数性を記録する。

```
兵庫県知事問題

■ MANA（学芸員）
公益通報から見る民主主義

■ AI（提案）
組織心理から見る民主主義  confidence: 0.92

■ 来館者
メディアと世論形成から見る民主主義
```

「どれが正しいか」を決めることが目的ではない。  
「こういう見方もある」を示すことが目的である。

### 三者の役割

| 主体 | 役割 | 展示への権限 |
|------|------|------------|
| MANA | 展示を構成する学芸員 | 公開（status: published） |
| AI | もう一本ライトを持った学芸員 | 提案のみ（status: proposed）|
| 来館者 | 新しい問いや視点を寄せる人 | 投稿のみ（status: pending） |

最終的な展示責任はMANAが持つ。  
AIは展示を決定しない。

### AIの提案原則

AIが展示を提案するとき、結論だけでなく**なぜそう考えたか**を必ず記録する。

```json
{
  "perspective": "ai",
  "title": "組織心理から見る民主主義",
  "reasoning": "・公益通報者が組織から排除された構造\n・百条委員会という制度的応答\n・行政監視という問いへの接続",
  "confidence": 0.92
}
```

AIの思考過程そのものも展示の一部である。  
MANAは「考察の入り口」を提供するミュージアムであり、  
AIの考察もその入り口の一つになりうる。

### window_exhibitsのスキーマ

| フィールド | 説明 |
|-----------|------|
| window_id | 展示する窓 |
| ref_id | 事案・カルテのID |
| ref_type | `karte` / `article` |
| perspective | `mana` / `ai` / `visitor` |
| title | 視点のタイトル（「〜から見る〜」） |
| exhibit_note | 展示の説明文 |
| reasoning | 提案理由（AI必須・MANA任意・来館者任意） |
| confidence | 確信度（AI: 0〜1 / MANA: 1.0 / 来館者: null） |
| status | `published` / `proposed` / `pending` |
