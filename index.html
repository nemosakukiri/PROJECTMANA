申請の流れ

生活保護は、申請すれば受けられる権利です。窓口に行くだけで申請できます。「相談」と「申請」は別物——「相談に来た」ではなく「申請に来た」と伝えることが大事です。

1
福祉事務所の窓口に行く
京都市の場合、各区の区役所内にある「生活福祉課」が窓口です。予約不要で行けます。一人で行くのが不安な場合は支援者や同行者と一緒に行っても構いません。
!
「申請したい」と最初に伝える
「相談したい」と言うと、申請させずに帰らせようとする場合があります。最初から「生活保護を申請したいです」と明確に伝えましょう。
2
申請書を受け取り、記入して提出
申請書は窓口でもらえます。その場で書いて提出できます。書けない部分は空欄でも受け付けてもらえます。「書類が揃わないと申請できない」は誤りです。
3
調査・訪問（申請から14日以内）
担当ケースワーカーが自宅訪問や資産調査を行います。調査には協力しますが、プライバシーに関わる不当な質問には答える義務はありません。
4
決定通知（申請から14日以内、最長30日）
開始または却下の通知が届きます。却下された場合は理由の説明を求め、不服があれば審査請求ができます。
覚えておいてほしいこと
申請日が保護開始日の基準になります。迷っている間も、申請は早いほど良いです。「申請書を渡さない」「書類が揃ってから来て」は違法な水際作戦です。
必要な書類

書類が揃っていなくても申請はできます。ただし、後から提出を求められる書類を事前に知っておくとスムーズです。

◎
申請時に持って行くと良いもの
身分証明書（健康保険証・マイナンバーカード等） / 印鑑 / 通帳（残高がわかるもの） / 家賃の分かる賃貸契約書 / 医療を受けている場合は診察券
△
後から提出を求められることがある書類
収入・資産がわかる書類 / 家族の状況に関する書類 / 医師の診断書（障害や疾病がある場合）
!
なくても申請できます
住所不定の方、書類を揃えられない方も申請できます。「書類が揃ってから来てください」と言われたら、その場で「申請書だけ受け取らせてください」と伝えましょう。
使える言葉

窓口では言い方ひとつで対応が変わることがあります。実際に効果があった言葉を集めました。

「相談があって来ました」
「生活保護を申請したいです。申請書をください。」
「働けないんですけど……」
「現在、生活に困窮しており、今日から申請したいと思っています。」
「どうすればいいですか」
「申請書を受け取る権利があると理解しています。書類が揃っていなくても申請はできるはずです。」
録音について
窓口での会話は録音できます。「記録のために録音させてください」と一言伝えて録音することで、後のトラブルに備えられます。相手の反応が変わることもあります。
窓口でよく言われること・その対処法

「水際作戦」と呼ばれる申請妨害のパターンと、その対処法です。

!
「まず家族に相談してから来てください」
家族への扶養照会は申請の条件ではありません。「家族への連絡は希望しません。申請は本人の権利として行いたいです」と伝えられます。DVや家族関係のトラブルがある場合は明示しましょう。
!
「まだ働けるでしょ」「若いから大丈夫」
職員が就労可能かどうかを窓口で判断する権限はありません。「申請書を受け取り、審査の中で判断していただきたいです」と伝えましょう。
!
「車を売ってから来てください」「保険を解約してから」
資産処分は申請後の審査の中で判断されるものです。窓口で条件を付けて申請を断ることは違法です。「申請を受け付けてください。資産については審査の中で説明します」と伝えましょう。

その日の日時・対応した職員の名前・言われた内容をメモする。支援団体や弁護士に相談する。後日、支援者と一緒に再訪する。

PROJECT MANAについて
このサイトは何か

PROJECT MANAは、行政・福祉・制度運用の現場で繰り返し起きている出来事を記録し、その背後にある構造的パターンを観測するための市民アーカイブです。

「制度はある。実装されていないだけだ。」という観点から、申請妨害・長期放置・記録改ざん・情報非公開など、制度が機能しない現場の事実を収集・分類・公開しています。

百葉箱と博物館

MANAは「百葉箱」と「博物館」の両方の性格を持っています。百葉箱が気象を観測するように、MANAは社会の構造を観測します。そして博物館が標本を分類・展示するように、MANAは収集した事案を整理・探索できる形で公開します。

記録することが目的であり、告発や運動の場ではありません。利用者が自分で調べ、自分で判断するための情報基盤を目指しています。

何を収録しているか

報道記事・行政資料・議会議事録・裁判例・当事者証言などをもとに、以下を収録しています。

  • 観測DB：全国の行政・福祉・制度に関する記事をAIが毎日自動収集・分類したデータベース
  • 事案カルテ：個々の事案を地域・分野・タグで整理した記録票
  • 用語辞典：制度用語の解説パネル（カルテやDBを読む際の参照用）
  • 法律辞典：関連法律のe-Govリンク集
運営について

PROJECT MANAは個人運営の市民アーカイブです。行政、福祉、権利擁護、地域社会などに関する公開情報を収集し、事案として整理・観測しています。

特定の政党・団体・企業とは関係がありません。

このサイトの限界について

PROJECT MANAは、答えを提示するためのサイトではありません。出来事を記録し、事案として整理し、社会構造を観測するための百葉箱です。

カルテやタグにはAIによる整理を含みます。誤りや不完全さが含まれる可能性があります。記録は完成品ではなく、観測の途中経過です。

観測に参加する

百葉箱は、設置した者だけが観測するものではありません。そこを通りかかった人が「今日は違う」と気づくことも、観測のひとつです。

カルテの誤り、関連する記事、気になるタグ、サイトへの違和感。どんな小さな気づきでも、PROJECT MANAの観測精度を高めます。

利用と引用について
PROJECT MANAは記事転載サイトではありません

PROJECT MANAは、報道記事・行政資料・公的情報をもとに事案を整理・分類・構造化したアーカイブです。記事の全文転載は行っておらず、原典を尊重し、可能な限り元記事・元資料へのリンクを掲載しています。

報道記事・行政資料等の著作権は、それぞれの権利者（報道機関・行政機関等）に帰属します。
